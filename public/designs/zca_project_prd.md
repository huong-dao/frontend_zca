# Project ZCA: Zalo Control & Automation - Product Requirements Document

## Overview
ZCA is a complete admin dashboard web application designed to manage Zalo accounts, groups, and automated messaging workflows.

## Visual Identity & Theme
- **Color Palette:** Primary Blue (#2563EB), Dark Navy Sidebar (#1E3A5F), Light Slate Background (#F8FAFC).
- **Typography:** Inter or Plus Jakarta Sans. Modern, clean, and legible.
- **Style:** Modern SaaS dashboard (Vercel/Linear style). Rounded corners (8px cards, 6px inputs), subtle shadows, white cards on light gray.

## Component Library
- **Buttons:** Primary, Secondary, Danger, Ghost, and Icon buttons with loading/disabled states.
- **Form Elements:** Styled Text/Password/Select inputs, Search with icons, and clear validation states.
- **Feedback:** Status Badges (Sent, Failed, Active, etc.) and Toast notifications.
- **Data Display:** Responsive tables with hover states, pagination, and empty states.
- **Layout:** Fixed dark sidebar (240px) with navigation groups and a light main content area.

## Screen Requirements
1. **Login Page:** Centered card, blue gradient background, email/password fields with validation.
2. **Dashboard:** Overview stats (Total Groups, Accounts, Messages) and a "Recent Messages" table.
